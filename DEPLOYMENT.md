# ICT/SMC Trading Bot - Deployed on Render 🚀

## 🌟 Features
- **24/7 Automated Signal Monitoring**
- **ICT/SMC Analysis** (FVG, Market Structure, EMA200)
- **AI-Powered Validation** (Groq Llama 3.1)
- **Multi-Asset Support** (Forex, Gold, Bitcoin)
- **Smart Alerts** (8/10+ confidence only)

## 📱 Commands
- `/start` - Main menu
- `/alerts` - Recent strong signals  
- `/monitor` - Toggle monitoring
- `/analyze [symbol]` - Custom analysis

## 🚀 Deployed on Render
- **Uptime**: 24/7
- **Data Source**: yfinance (reliable cloud data)
- **Monitoring**: Every 5 minutes
- **Cooldown**: 30 minutes between signals
