# Render Deployment Configuration

## Environment Variables (Set in Render Dashboard):
- TOKEN: Your Telegram Bot Token
- GROQ_API_KEY: Your Groq API Key
- MT5_LOGIN: Your MT5 Login (optional)
- MT5_PASSWORD: Your MT5 Password (optional) 
- MT5_SERVER: Your MT5 Server (optional)

## Deployment Steps:
1. Connect GitHub repository to Render
2. Create Web Service
3. Set environment variables in Render dashboard
4. Deploy - Bot will run 24/7

## Bot Features:
- 24/7 automated signal monitoring
- ICT/SMC analysis for forex, gold, crypto
- AI-powered trade validation
- Automatic strong signal alerts
- MT5 data source with fallback

## Monitoring:
- Checks 5 symbols every 5 minutes
- Alerts for 8/10+ confidence setups
- 30-minute cooldown between signals
